OSVVM
=====

`Open Source VHDL Verification Methodology <https://github.com/OSVVM/OSVVM>`__ writes test results as YAML files for its
internal data model storage. Some YAML files are written by the VHDL code of the verification framework, others are
written by `OSVVM-Scripts <https://github.com/OSVVM/OSVVM-Scripts>`__ as a test runner.
